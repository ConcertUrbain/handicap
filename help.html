# Pour changer l'image d'une question (icone dans le menu en haut)

Il faut rajouter une image dans le dossier images/question_icons/

images/question_icons/qst-[ID].png

Les images sont contraintes à une dimension carrée de 50px 50px.

# Checklist pour faire fonctionner une question

- Une question doit avoir un titre

- Une question doit avoir une description. Cette description sera affichée dans la page de présentation.

- Une question doit avoir une vidéo d'illustration (mme jeanvoine) associée. Cette vidéo est marqué par des "" dans son titre.

# Changer la vidéo dans la rubrique contexte de la page de présentation

- Changer le constante PRESENTATION_VIDEO dans const.php
